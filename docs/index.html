PARALLEL WORLD

作品概要

HEWと呼ばれる学内コンテストの一年次にて銀賞を獲得した作品です。
2Dと3Dの世界を行き来する独創的なゲームシステムが特徴で、視点切り替えによるパズル要素を実装しました。
C++のコンソールにて制作しました。

担当・工夫した点
  • 2D / 3D の視点切り替えシステムの設計・実装
  • コンソール上でのレンダリング処理の最適化
  • プレイヤーの直感的な操作を実現するキャラクター制御
  • パズル要素のバランス調整とレベルデザイン
制作期間・規模
